投资需要加强（非常重要）的3点能力：
1：hold的住现金的能力。
2：不乱投资。
3：识人头。
以上3点是相辅相成的，今天说说第1点：

说说投资需要加强的3点能力里的第2点：不乱投资。

投资的最基础心法是：谋定而后动。
很多人嘴上的投资其实不是投资，那叫跟风。拿群主相对擅长的啃房来说：周围99%以上的人在群主看来是：盲从+傻啃。具体表现在：人云亦云，没有自己的分析见解，容易被他人洗脑。（这里多说1句：洗脑其实也没关系，关键要找对洗脑的人）

每个人生来的经济基础不一样是客观存在的前提，也就是起跑线不一样。起跑线不行的TX要做的最重要的事情，就是正确的评估自己和制定切实可行的计划，而不是盲目的发老急去showhand，那种赢了嫩模，输了干活的思路千万要不得。

自由经济与利润率

发表于 2018-12-17 | 更新于 2018-12-21

自由市场带来价格趋同于价值

在一个自然自由竞争机制充分发挥作用的经济体系中，产品的市场价格必然会与自然价格相一致，每一种要素的市场价格也会和它们的自然价格相一致。

因为只要有一种行业、产品出现超额利润，就会有其他资本、资源跟进，进而拉低整体利润率，直至趋同于社会平均利润率。最简单的例子就是卖煎饼。王大娘街边卖煎饼月入3万，那么立马会引来李大娘、张大娘、赵大娘，他们充分竞争，最终每人摊薄利润，可能每人月入5千甚至3千2千，然后有人退出竞争，达到一种平衡状态，也就拉到了社会平均利润率。

管制带来价格脱离价值

很明显我们生活的市场并不是一个完全自由的市场，既然不是自由市场就必然会引起市场价格偏离于自然价格。因为管制限制了供给的自由流动，要么供过于求引起价格下跌，要么供不应求造成价格上涨。拿管制最严重的地产市场来说再明了不过。比如重庆特殊的地票制度保证市场土地供需基本平衡，北京等地的供地政策明显偏向于收紧，这就造成了重庆房价的长期低迷和北京等城市的长期上涨。

完全自由市场不会完全到来，只能无限的接近

因为存在管制，所以不会出现完全自由的市场经济体。那么就会影响到供求关系，进而影响到商品的市场价格。管制造成了某些商品价格偏离自然价格很远，这就存在了套利空间。当然这些套利空间有的向民众开放，有的不开放，开放的部分需要人们去发现。找到这部分，参与其中，在被大众涌入拉低到平均利润之前推出，就能获得超额利润。

以本守之，以末致财

所以，要想获得财富，第一要有本钱，第二要去发现套利空间。正常工作赚取普通回报就是本，这是行动的基础，然后发现机会迅速做大，这是末。平时蛰伏，寻找泡沫，在泡沫起来之前提前潜入建仓，在泡沫泛滥破裂之前逃走，这是最理想的模型。对照生活中具体的场景，相信你会很有体会。

69. x 的平方根

题目

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

示例 1:

输入: 4
输出: 2
示例 2:

输入: 8
输出: 2
说明: 8 的平方根是 2.82842…,
由于返回类型是整数，小数部分将被舍去。

分析

这个是求开方的结果，很容易想到按顺序i找，如果存在 i 就去找

public int mySqrt(int x) {
if (x <= 0) {
return 0;
}

    for (int i = 0; i < x; i++) {
        boolean flag = i*i <= x && (i + 1) * (i + 1) > x;
        if (flag) {
            return i;
        }
    }

    return 0;

}

boolean flag = (long) i (long) i <= x && (long) (i + 1) (long) (i + 1) > x;

public static int mySqrt(int x) {

    if (x <= 0) {
        return 0;
    }
    int max = x / 2 + 1;
    int min = 0;
    while (min <= max) {
        int mid = min + (max - min) / 2;
        long rs = (long) mid * (long) mid;
        if (rs == x) {
            return mid;
        }

        if (rs < x) {
            min = mid + 1;
        } else {
            max = mid - 1;
        }
    }

    return max;

}
